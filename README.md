## About

Most applications work in demos. Fewer survive real users, real data, and real business pressure.

I'm a Full-Stack Engineer with **3+ years** shipping production systems — not prototypes. I've worked on platforms where a broken API meant a broken business, and where a bad architecture decision meant a six-month rewrite. That context shapes how I write code.

What I actually do:
- Build frontend systems in **React / Next.js** that scale beyond the first release
- Design **Node.js / Python backends** with database-backed business logic that's built to be maintained
- Deploy on **AWS** with real CI/CD — not just "works on my machine"
- Integrate **AI-powered features** that solve real user problems
- Mentor engineers — taught **500+ students**, mentored globally at **Stanford Code in Place**

Currently deepening expertise in **cloud architecture**, **system design**, and **advanced state management patterns**.
